# Recipe Finder App

A web application that allows users to search for different recipes and view details such as ingredients, images, and cooking instructions.

This project was developed to practice JavaScript concepts including DOM manipulation, API integration, and asynchronous programming.

## Features

- Search recipes by food name
- Fetch recipe data from an external API
- Display recipe images and details
- View ingredients and cooking instructions
- Dynamic content updates based on user input
- Responsive user interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API
- Async/Await
